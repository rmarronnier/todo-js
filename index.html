Todo list

L'objectif final de cette série de TD est de créer une application de gestion de tâches. Ce fil rouge vous permettra d'aborder les bases du Javascript.

Principes généraux

  1. Les variables
  2. Les objets
  3. Les fonctions

Modifier le fichier src/todo.js pour :

  1. Déclarer une variable contenant la valeur "Faire mon CV" et l'afficher dans la console du navigateur.
  2. Déclarer une class Task comprenant les propriétés suivantes : nom de la tâche, son statut (réalisé ou non) et sa priorité (valeur entier de 1 à 5).
  3. Créer un tableau contenant quatre objets de type Task dont les valeurs de priorité sont différentes.
  4. Afficher dans la console les noms des tâches contenues dans le tableau.
  5. Changer la valeur des deux tâches les plus urgentes (priorité > 4) en "réalisée"
  6. Afficher dans la console les titres des tâches non réalisées

Manipulation du DOM (sans JQuery !)

  1. Créer un fichier todo.html avec un markup de base et contenant un formulaire de création d'une nouvelle tâche. Pointer cette page vers le fichier src/todo.js
  2. Créer un fichier todo.html avec un markup de base et contenant un formulaire de création d'une nouvelle tâche.
